**PUT /chains/:name:**

**DELETE /chains/:name:**
Replace configRef reads with admin.getConfig() for the existence check and default chain guard. Solution: add `defaultChain: string` to `AdminRouteDeps` and pass `config.settings.defaultChain` from index.ts. This is cleaner than exposing settings through the repository.

const { chains: chainConfigs } = admin.getConfig();
